
Voxel Engine
============

My nameless engine used for rendering cubic voxel meshes, with [SGL](https://github.com/nnarain/sgl-wrapper)
at its core.


Dependencies
------------

- SGL
- GLFW
- luabind
	- lua 5.2
	- Boost C++
